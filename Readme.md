# 🎨 Random Color Generator

A simple and interactive web app that generates random colors instantly.<br>
Perfect for designers, developers, or anyone looking for color inspiration.

# 🚀 Features

🎲 Generate random colors with a single click <br>
📋 Get the RGB color combination <br>
🌈 Real-time background color updates <br>
⚡ Lightweight and fast <br>

# 🛠️ Built With

HTML5 <br>
CSS3 <br>
JavaScript (Vanilla JS) <br>
